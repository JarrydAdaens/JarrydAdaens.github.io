What It Does

Kung-Fu is intended for developers who work with local code folders and agentic IDE artifacts.

Local workspace helper

Works with folders you configure on your own device, so project context stays close to the files you already use.

No account required

The app is designed to run without user accounts, subscriptions, or cloud synchronization.
