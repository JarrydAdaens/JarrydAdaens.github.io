App

Kung Fu

Kung-Fu: Agentic IDE Helper is a local-first desktop utility for keeping agentic IDE workspace files, notes, and artifacts easier to find across configured folders.

Summary

A local-first helper for developers working with agentic IDE artifacts and configured workspace folders.

Status

First app intended for publication.

What It Does

Kung Fu helps organise local workspace artifacts so project notes and agent-related files are easier to find beside the code they belong to.

Why It Exists

Agentic IDE workflows can produce useful context that becomes scattered. Kung Fu exists to keep that context closer to the working folders.

Platforms

Windows desktop.
